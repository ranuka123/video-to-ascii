# video-to-ascii
A program that outputs video frames in ascii to terminal using the ffmpeg api
```
make
./test video.avi
```
<img src="https://github.com/snjt/video-to-ascii/blob/master/demo.gif" width="200" height="200">
# Dependencies
- ffmpeg
- ncurses
- gcc
```
Tested on OSX 10.8.4, gcc 4.9.2
```
