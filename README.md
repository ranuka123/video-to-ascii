# video-to-ascii
A program that outputs video frames in ascii to terminal using ffmpeg
<br />
To use:
```
make
./test video.avi
```
![alt tag](https://github.com/snjt/video-to-ascii/blob/master/demo.gif)
# Dependencies
ffmpeg, ncurses
